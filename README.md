# Sistema de Contratación CAS - UGEL Talara

![Estado del Proyecto](https://img.shields.io/badge/Estado-Finalizado-success)
![Tecnologías](https://img.shields.io/badge/Stack-React%20%7C%20Node.js%20%7C%20MySQL-blue)

Este sistema web ha sido desarrollado para automatizar y optimizar el proceso de **Contratación Administrativa de Servicios (CAS)** en la Unidad de Gestión Educativa Local (UGEL) Talara. El objetivo principal es digitalizar la gestión de expedientes, eliminando el uso excesivo de papel y mejorando los tiempos de respuesta institucional.

## 📋 Características Principales

- **Gestión de Postulantes:** Registro seguro, carga de currículum vítae y anexos en formato PDF.
- **Seguimiento en Tiempo Real:** Los usuarios pueden ver el estado de su postulación (Recibido, En Revisión, Calificado, etc.).
- **Panel Administrativo:** Herramientas para que el equipo de Recursos Humanos califique expedientes y genere reportes automáticos.
- **Seguridad:** Autenticación de usuarios y protección de datos sensibles.
- **Sostenibilidad:** Reducción del impacto ambiental mediante la digitalización de procesos administrativos.

## 🛠️ Stack Tecnológico

- **Frontend:** React.js
- **Backend:** Node.js
- **Base de Datos:** MySQL / MariaDB
- **Entorno de Desarrollo:** XAMPP / Visual Studio Code
- **Arquitectura:** Modelo-Vista-Controlador (MVC)

## ⚙️ Requisitos del Sistema

- Navegador web actualizado (Chrome, Firefox, Edge).
- Conexión a internet para el acceso remoto.
- Servidor con soporte para aplicaciones Node.js y base de datos relacional.

## 📂 Estructura del Proyecto

- `/client`: Código fuente de la interfaz de usuario (React).
- `/server`: Lógica del servidor, API REST y conexión a base de datos (Node.js).
- `/docs`: Documentación técnica y manuales de usuario.
